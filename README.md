# Campus Lost & Found Portal

A web-based platform to help students, faculty, and staff report and recover lost and found items on campus efficiently.

---

## Problem Statement
Campus lost and found systems are scattered and informal, making it difficult to recover misplaced belongings quickly and reliably.

---

## Our Solution
We built a centralized, real-time Campus Lost & Found Portal that allows users to report lost and found items and view listings instantly. The platform uses Google technologies to ensure reliability, scalability, and intelligent summaries.

---

## Features
- Report lost and found items with details
- View real-time listings
- AI-generated item summaries using Google Gemini
- Search and filter items
- Mark items as resolved
- Live hosted MVP

---

## Google Technologies Used
- Firebase Firestore (real-time database)
- Firebase Hosting (deployment)
- Google Gemini AI (AI-powered summarization)

---

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in a browser
